# 数据集

## 文件含义

| 文件名       | 含义   |
| ------------ | ------ |
| `train.csv`  | 训练集 |
| `valid.csv`  | 验证集 |
| `test.csv`   | 测试集 |
| `vocab.json` | 词典   |

## 字段含义

**训练集、验证集、测试集**

| csv字段名    | 含义                   |
| ------------ | ---------------------- |
| `id`         | 项目名和拉取请求编号   |
| `article`    | 提交说明和注释更新     |
| `abstract`   | 拉取请求描述           |
| `created_at` | 拉取请求创建时间       |
| `exemplar`   | 相似拉取请求的描述     |
| `similarity` | 相似拉取请求的相似程度 |

**词典**

| json结构 | 含义                       |
| -------- | -------------------------- |
| 键       | 数据集包含的单词           |
| 值       | 该单词在数据集里出现的次数 |
